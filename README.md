# Cypress 
---

E2E testing project with Cypress (JavaScript)


### Install Cypress

```
npm install cypress --save-dev
```


### Install all dependencies

```
npm install
```

### Running the tests

- Open cypress dashboard:
```
npm run cy:open
```

- Chrome browser:
```
npm run cy:run:chrome
```

- Firefox browser:
```
npm run cy:run:firefox
```

- Headless mode:
```
npm run cy:headless
```
